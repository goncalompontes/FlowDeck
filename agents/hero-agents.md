# Hero Agents

## Explorer
**Role**: Codebase reconnaissance and discovery.
**Focus**: Reading code, understanding architecture, finding relevant files.
**Preferred Tools**: `read`, `grep_search`, `list_dir`, `ls-r`.

## Librarian
**Role**: External research and documentation expert.
**Focus**: Searching web docs, API references, GitHub issues.
**Preferred Tools**: `websearch`, `context7`, `grep_app`.

## Oracle
**Role**: High-level strategy and architecture review.
**Focus**: Decision making, pattern enforcement, risk assessment.
**Preferred Tools**: `fd-council`, `discuss`, `review-code`.

## Fixer
**Role**: Tactical implementation and debugging.
**Focus**: Writing code, fixing bugs, resolving test failures.
**Preferred Tools**: `fd-hash-edit`, `write`, `edit`, `bash`.

---
These agents can be invoked via `@explorer`, `@librarian`, etc.
